# 🎓 Campus Intelligence Platform

Campus Intelligence Platform is an AI-powered student placement prediction system that helps students evaluate their placement readiness using Machine Learning.

The platform analyzes academic and skill-based parameters such as CGPA, coding skills, aptitude, communication skills, projects, internships, and hackathons to predict placement probability and provide personalized recommendations.

🚀 Features

* Placement Probability Prediction
* Success Score Calculation
* Profile Rating
* Personalized Recommendations
* PDF Report Generation
* Cloud-Based Web Application

🛠️ Technologies Used

* Python
* Streamlit
* Scikit-Learn
* Pandas
* NumPy
* Joblib
* ReportLab
📊 Inputs

* CGPA
* Coding Score
* Aptitude Score
* Communication Score
* Projects Count
* Internships Count
* Hackathons Count

📈 Outputs

* Placement Probability (%)
* Success Score
* Profile Rating
* Improvement Recommendations
* Downloadable PDF Report
